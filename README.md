# thisweb-skills

Reusable agent skills for website work.

This repository contains reusable installable skills:

- `web-brand-md`: clarifies brand direction before website planning and turns it into a concise `BRAND.md` for downstream AI workflows.
- `design-md`: generates strict Tailwind `DESIGN.md` files for AI-assisted interface work.
- `better-ui`: improves interface polish, component behavior, typography, spacing, motion, and visual quality.
- `logo-generator`: generates SVG logo concepts, PNG exports, HTML logo showcases, and Codex imagegen-based brand presentation images.

## Install

Install the whole repository:

```bash
npx skills add https://github.com/kun1225/thisweb-skills
```

Install only one skill:

```bash
npx skills add https://github.com/kun1225/thisweb-skills --skill web-brand-md
npx skills add https://github.com/kun1225/thisweb-skills --skill design-md
npx skills add https://github.com/kun1225/thisweb-skills --skill better-ui
npx skills add https://github.com/kun1225/thisweb-skills --skill logo-generator
```
