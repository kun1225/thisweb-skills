# thisweb-skills

Reusable agent skills for website work.

This repository currently contains one installable skill:

- `website-brand-guideline`: clarifies brand direction before website planning and turns it into a concise `BRAND.md` for downstream AI workflows.

## Install

Install the whole repository:

```bash
npx skills add https://github.com/thisweb/thisweb-skills
```

Install only this skill:

```bash
npx skills add https://github.com/thisweb/thisweb-skills --skill website-brand-guideline
```
